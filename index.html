Navegando de forma anonima

Introdução

Para navegar na internet você precisa utilizar um provedor de acessos, comumente chamado de ISP (do inglês internet service provider). No brasil os principais são NET, GVT, OI, Telefônica etc.

Sempre que você se conecta seu ISP te designa um endereço IP (ou simplesmente IP). Esse endereço serve para você se comunicar com outros computadores e servidores na Internet. Quando você digita google.com.br no seu navegador seu computador consulta um servidor de DNS que o informa para qual endereço IP esse domínio está "apontando". No caso do google.com.br o IP é: 74.125.234.24.

Introdução

Quando você se conecta a um servidor seu endereço IP é exposto. Seja:

  • navegando em um website
  • baixando um arquivo
  • utilizando um programa de mensagens instantânes
  • fazendo login em uma rede social
  • utilizando um serviço de emails

Mesmo que seu IP possa ser relativamente randômico e dependenda da designação do seu provedor de acessos, essas empresas matêm registros (logs) de qual IP cada um de seus usuários estava utilizando, com dia e horário.

Para ver seu IP atual acesse: http://qualmeuip.com.br/

TOR

  • O primeiro passo para navegar anonimamente é conseguir um IP diferente através de um PROXY
  • A maneira mais fácil e barate de se conseguir isso é utilizando a rede TOR.
  • Tor é um software e uma rede open source que utiliza rotas randômicas através de seus nós.
  • Todo o dado que trafega pela rede é criptogrado e não rastreável
  • Mais informações em https://www.torproject.org/

Passo a passo

  • Baixar o Tor Broser Bundle aqui
    • O Bundle contém uma versão "portátil" do Firefox já configurada para funcionar com a rede
  • Após a instalação inicie o aplicativo "Start Tor Browser.exe" na pasta que foi criada
    • Você será conectado automaticamente à rede e o Firefox portátil (anonimizado) será inicializado
    • No programa ainda existem outras opções, como "mudar de identidade", útil quando se que trocar o "ip final"

Fique anônimo de verdade!

Para garantir sua privacidade é necessários seguir alguma recomendações importantes:

  • Use o browser do bundle
  • Não instale plugins no browser
    • Flash e similares vêm desabilitados por padrão, uma vez que podem ser manipulados de forma a revelar seu real IP
  • Use HTTPS quando disponível
    • Isso garante uma conexão criptografada de ponta-a-ponta.
    • Dessa forma seus dados ficam critgrafados inclusive para seu provedor de acesso e eventuais redes que você esteja utilizando.
  • Não abra documentos baixos pelo TOR enquanto estiver online
    • NÃO IGNORE ESSE AVISO. Isso vale especialmente para arquivos .DOC e .PDF
  • Use um tor bridge.
    • O tor não previne que outras pessoas que têm acesso a sua rede saibam que você está usando a rede Tor

Não negligencie seu passado

  • Nunca, NUNCA logue com seu navegador "anonimizado" em contas que você tenha utilizado previamente sem ele
    • Gmail, Youtube etc mantêm um LOG acessos.
    • Fazendo cruzamento de dados é possível descobrir sua identidade
  • Isso siginifica que você terá que criar contas novas, não "infectadas"

    • Opções pagas

    Existem também proxyes pagos, que geralmente funcionam através de VPNs. Nesse caso, todo o seu tráfego é feito através do Proxy. Diferente do Tor que inicialmente fica restrito ao browser habilitado.

    Esse tipo de mecanismo é muito utilizado para acessar conteúdos que apresentam restrições regionais (filmes disponíveis apenas para quem acessa dos EUA etc.)

    Alguns proxyes com foco na privacidade que podemos citar são:

    • Privacy.io
    • ProXPN
    • IP Vanish

    Atenção: algumas empresas que oferecem o serviço de Proxy também mantém logs. Pesquise e nunca ache que você pode se esconder atrás de um proxy para comenter crimes.

    Enjoy your privacy!
